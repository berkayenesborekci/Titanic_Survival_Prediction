# Titanic Survival Prediction
A machine learning project predicting Titanic passenger survival using the Kaggle dataset.

## What I did
- Explored and cleaned the dataset (handled missing Age, Cabin, Embarked values)
- Encoded categorical features (Sex, Embarked)
- Trained a Random Forest Classifier
- Visualized feature importances to understand which factors affected survival most

## Results
- Training accuracy: 97.9%
- Most important features: Sex, Fare, Age

## Tech Stack
- Python, pandas, numpy, scikit-learn, matplotlib, seaborn

## Dataset
[Kaggle Titanic Competition](https://www.kaggle.com/competitions/titanic)
